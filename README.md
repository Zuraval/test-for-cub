# Order Management Dashboard

## Запуск

npm install
npm run dev

## Стек

Vite, TypeScript (strict), Ant Design v5, Zustand, Axios

## Стейт-менеджер Zustand

Минимальный boilerplate, полная типизация, нет лишних ре-рендеров
